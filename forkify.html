Resume

Forkify app in which we can search for a recipe and also add our recipe if we which.

This website uses and API in order to search and add add new recipe. More information on https://forkify-api.herokuapp.com/.

In this Forkify app I put in practice all the ES6 Javascript features such as arrow function, working with modules, Fetch requests.
