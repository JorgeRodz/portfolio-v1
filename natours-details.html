Resume

Landing page project in which a fictional enterprise are offering tours mainly focussing on natural destinations.

In this project I put all the knowledge bout HTML and CSS, focusing on semantic HTML and CSS animations and transitions. In also work with SASS CSS approach to make reusable pieces of code and to have a clean and organize project folders. And to pack all the code together and have a minimal code usage when making the deploy I used parcel bundler.
